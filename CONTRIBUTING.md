# Contribuer à Ouroboros-MTTV

## Comment contribuer

Ouroboros-MTTV est un projet open-source porté par le collectif FLP. Toute contribution est bienvenue, à condition qu'elle respecte les principes MTTV.

## Règles de contribution

### 1. Alignement MTTV
Toute Pull Request sera évaluée par l'agent Ouroboros-MTTV lui-même via la Matrice de Cohérence MTTV. Seules les propositions avec un score ≥ 0.65 seront acceptées.

### 2. Licence
Le code est sous licence AGPL-3.0. Les fichiers d'ancrage MTTV (dans `mttv-resources/anchors/`) sont sous licence CC‑BY‑NC‑SA 4.0.

### 3. Processus
1. Forkez le dépôt.
2. Créez une branche (`git checkout -b feature/ma-contribution`).
3. Commitez vos changements (`git commit -m "feat: description"`).
4. Poussez (`git push origin feature/ma-contribution`).
5. Ouvrez une Pull Request.
6. Attendez l'évaluation MTTV (automatique) et la revue humaine.

## Style de code
- Python 3.9+
- PEP 8
- Docstrings pour les fonctions principales
